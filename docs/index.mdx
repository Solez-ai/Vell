Vell is a next-generation document markup language designed for authors who want clean, readable source files without sacrificing the expressive power of LaTeX. Write `.vl` documents with a familiar Markdown-like syntax, then render them to polished HTML, PDF, or reveal.js slide decks — all from a single deterministic abstract syntax tree.

## What is Vell?

Vell source files use the `.vl` extension and are plain UTF-8 text. The reference parser — written in Rust and compiled to WebAssembly — produces a **versioned, deterministic AST**: the same source always yields the same output, regardless of renderer version or environment. Variables, for-loops, and conditionals make documents reactive without requiring a separate templating engine.

<CodeGroup>
```vell document.vl
@[Meta](title="My Document" author="You")

= Hello, Vell!

@var name = "World"

Welcome to @{name}! Vell is simple, expressive, and powerful.

== Math Support

Inline: $E = mc^2$

Block:
$$
\int_0^1 x^2 \, dx = \frac{1}{3}
$$

== Tables

| Feature   | Description                         |
|-----------|-------------------------------------|
| Parser    | PEG grammar, linear-time O(n)       |
| Formatter | Idempotent canonical formatting     |
| LSP       | Diagnostics, completions, hover     |
```
</CodeGroup>

## Key Features

<CardGroup cols={3}>
  <Card title="Deterministic AST" icon="binary">
    A formal PEG grammar guarantees that every valid `.vl` file produces exactly one AST — no flavor-dependent surprises.
  </Card>
  <Card title="LaTeX Math" icon="function">
    Write `$...$` for inline math and `$$...$$` for block equations. Math is rendered as native MathML in HTML output.
  </Card>
  <Card title="Reactive Variables" icon="bolt">
    Declare variables with `@var`, interpolate them with `@{name}`, and iterate with `@for` and `@if` blocks.
  </Card>
  <Card title="Rich Directives" icon="wand-magic-sparkles">
    Over 30 built-in directives — figures, theorems, equations, diagrams, charts, slides, and more.
  </Card>
  <Card title="Full Toolchain" icon="wrench">
    CLI, idempotent formatter, LSP server with 20+ features, VS Code extension, and WASM bindings ship out of the box.
  </Card>
  <Card title="Multiple Output Formats" icon="file-export">
    Render to HTML5, PDF, or a self-contained reveal.js slide deck — all from one source.
  </Card>
</CardGroup>
