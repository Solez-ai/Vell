Vell treats mathematics as a first-class citizen. You write LaTeX expressions directly in your source file, and the HTML renderer converts them to MathML — the web standard for accessible, scalable math. Whether you need a quick inline formula or a numbered theorem with a proof and cross-references, Vell has dedicated syntax for every case.

## Inline Math

Surround a LaTeX expression with single `$` delimiters to render it inline within a paragraph:

```vell
Einstein's famous equation $E = mc^2$ relates mass and energy.

In a right triangle, $a^2 + b^2 = c^2$ always holds.

Euler's identity $e^{i\pi} + 1 = 0$ unites five constants.

The standard deviation is $\sigma = \sqrt{\frac{1}{n}\sum_{i=1}^n (x_i - \mu)^2}$.
```

Inline math renders inside the text flow at the same baseline as surrounding prose.

---

## Block Math

Surround a LaTeX expression with `$$` delimiters — each on its own line — to render a centered display-mode equation:

```vell
$$
\int_0^1 x^2 \, dx = \frac{1}{3}
$$

$$
\frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$

$$
\int_{-\infty}^{\infty} e^{-x^2} \, dx = \sqrt{\pi}
$$
```

Content between `$$` markers is raw LaTeX and is not parsed as Vell inline markup.

<Note>
Both the opening `$$` and closing `$$` must be on their own lines. A `$$` that appears in the middle of a line is not treated as a math block delimiter.
</Note>

### Stacked Subscripts

Use `\substack` to stack multiple lines in a subscript position, typically for sums over multiple conditions:

```vell
$$

```

### Sideset and Prescript

Place scripts on the **left** side of an operator with `\prescript`:

```vell
$\prescript{a}{b}{C}$    (left superscript a, left subscript b)
$\prescript{1}{2}{X}_3^4$  (combined left and right scripts)
```

For large operators like `\sum` and `\prod`, use `\sideset` to place prescripts and postscripts around the operator while preserving its limits:

```vell
$$
\sideset{_a^b}{_c^d}\sum_{n=1}^{\infty} a_n
$$

$$
\sideset{}{^\ast}\prod_{k=1}^N x_k
$$
```

## Theorem Environments

Vell includes dedicated directives for mathematical writing: theorems, proofs, lemmas, corollaries, definitions, and more. Each numbered environment auto-increments its counter document-wide.

### @[Theorem]

```vell
@[Theorem](label="thm:pythagoras" name="Pythagoras") {
  For any right triangle with legs *a*, *b* and hypotenuse *c*:

  $$
  a^2 + b^2 = c^2
  $$
}
```

### @[Proof]

```vell
@[Proof]() {
  Consider a square of side $a + b$. Its area is $(a + b)^2$.
  Arrange four copies of the triangle inside — the remaining area
  equals $c^2$. Expanding $(a+b)^2 = a^2 + 2ab + b^2$ and
  subtracting the four triangles (area $2ab$) gives $a^2 + b^2 = c^2$.
}
```

### @[Lemma] and @[Corollary]

```vell
@[Lemma](name="Triangle Inequality") {
  For any triangle with sides *a*, *b*, *c*:

  $$
  a + b \geq c
  $$

  with equality only in the degenerate case.
}

@[Corollary](name="Reverse Triangle Inequality") {
  For any triangle with sides *a*, *b*, *c*:

  $$
  |a - b| \leq c
  $$
}
```

### Other Math Environments

```vell
@[Definition](name="Prime Number") {
  A natural number *p* > 1 is *prime* if its only positive
  divisors are 1 and *p* itself.
}

@[Example]() {
  The prime numbers less than 20 are: $2, 3, 5, 7, 11, 13, 17, 19$.
}

@[Remark]() {
  The number 1 is neither prime nor composite by convention.
}

@[Axiom](name="Euclid's Fifth Postulate") {
  If a line segment intersects two straight lines forming two
  interior angles on the same side summing to less than two right
  angles, the two lines meet on that side when extended.
}

@[Conjecture](name="Goldbach's Conjecture") {
  Every even integer greater than 2 can be expressed as the
  sum of two primes.
}

@[Proposition](name="Sum of First n Naturals") {
  For any positive integer *n*:

  $$
  \sum_{k=1}^{n} k = \frac{n(n+1)}{2}
  $$
}
```

## Numbered Equations and Cross-References

Use `@[Equation]` to create a standalone, auto-numbered equation with an optional label:

```vell
@[Equation](label="e:mass-energy" source="E = mc^2")

@[Equation](source="\int_{-\infty}^{\infty} e^{-x^2} \, dx = \sqrt{\pi}")

@[Equation](label="e:pythagoras" source="a^2 + b^2 = c^2")
```

Then reference any labeled equation or theorem with `@[Ref]`:

```vell
As shown in equation @[Ref](label="e:mass-energy"), mass and energy
are interchangeable.

Theorem @[Ref](label="thm:pythagoras") proves this geometric identity.
```

The renderer resolves references in a two-pass pipeline:
1. **First pass** — collects all labels and computes auto-incremented numbers.
2. **Second pass** — replaces each `@[Ref]` with a clickable link showing the resolved number (for example, `(1)` for an equation or `Theorem 1 (Pythagoras)` for a theorem).

If a referenced label does not exist, the renderer displays `[?label]` in red italic text.

## MathML Output

The HTML renderer converts all LaTeX math to MathML — the W3C standard for web mathematics. MathML renders natively in all modern browsers without JavaScript, scales correctly with page zoom, and is fully accessible to screen readers.

```vell
Inline: $E = mc^2$

$$
\sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{\pi^2}{6}
$$
```

Produces roughly:

```html
<!-- Inline -->
<math xmlns="http://www.w3.org/1998/Math/MathML">
  <mi>E</mi><mo>=</mo><mi>m</mi><msup><mi>c</mi><mn>2</mn></msup>
</math>

<!-- Block -->
<div class="vell-math-block">
  <math xmlns="http://www.w3.org/1998/Math/MathML" display="block">
    <!-- … MathML for the sum … -->
  </math>
</div>
```

<Tip>
MathML requires no external JavaScript library. Unlike MathJax or KaTeX, the Vell HTML renderer produces self-contained output — the math is ready to display the moment the page loads.
</Tip>

### KaTeX Polyfill for Older Browsers

For browsers that do not support MathML natively (e.g., older Chrome versions, some mobile browsers), the renderer includes an automatic KaTeX polyfill:

1. On page load, a small inline script tests MathML support by rendering a test `<math>` element and checking its rendered height.
2. If MathML is **not** supported, the script loads KaTeX (CSS + JS) from CDN.
3. KaTeX re-renders all `<math>` elements on the page using their `data-latex` attribute values.

This polyfill is **automatic and transparent** — no configuration or extra JavaScript is needed. The KaTeX CDN script is only loaded when the browser actually needs it.

## Complete Math Example

```vell
= Fundamental Theorem of Calculus

@[Theorem](label="thm:ftc" name="Fundamental Theorem of Calculus") {
  Let *f* be a continuous real-valued function on a closed interval. Then:

  @[Equation](source="\frac{d}{dx} \int_a^x f(t) \, dt = f(x)")

  @[Equation](source="\int_a^b f(x) \, dx = F(b) - F(a)")
}

@[Proof]() {
  By the definition of the derivative:

  @[Equation](source="F'(x) = \lim_{h \to 0} \frac{1}{h} \int_x^{x+h} f(t) \, dt = f(x)")
}

As shown in @[Ref](label="thm:ftc"), differentiation and integration
are inverse operations. The inline form $\int_a^b f(x)\,dx = F(b) - F(a)$
is one of the most important results in mathematics.
```
